What is N-Back?

The n-back test is a cognitive task used to measure working memory. You'll see a sequence of stimuli and need to identify when the current item matches the one from N positions back.

ðŸ“‹

How to Play

  • Stimuli will appear one at a time
  • Press SPACE or click MATCH when the current stimulus matches the one from N positions back
  • For example, in a 2-back test, if you see: A, B, A - press MATCH on the second A
  • Be quick but accurate!

Scoring

Your score is based on accuracy (hits, misses, false alarms, and correct rejections). Higher scores indicate better working memory capacity.
